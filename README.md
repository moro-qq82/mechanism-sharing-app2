# セットアップ補助ツール

このリポジトリには、開発環境を素早く把握するためのスクリプト `scripts/bootstrap.sh` を追加しました。Node.js / Python / Docker など複数スタックを横断的に検出し、推奨される初期コマンドを日本語で提示します。実際のインストールやサーバー起動の前に一度実行しておくと、別環境での立ち上げ漏れを減らせます。

## 使い方

1. リポジトリのルートで以下を実行します。
   ```bash
   ./scripts/bootstrap.sh
   ```
2. スクリプトが検出した推奨コマンドを上から順番に実行します。
3. `.env.example` などが案内された場合は、指示に従って環境変数ファイルを準備します。
4. `package.json` などの存在に応じて、依存関係のインストール・マイグレーション・テストコマンドが表示されます。

> **注意:** `jq` など一部の補助ツールが未導入の場合、手動で `package.json` 等を確認する旨のメッセージが表示されます。その場合は案内に従ってツールをインストールするか、ファイルを直接確認してください。

より詳細な手順や環境別の補足は [`docs/setup_guide.md`](docs/setup_guide.md) を参照してください。
